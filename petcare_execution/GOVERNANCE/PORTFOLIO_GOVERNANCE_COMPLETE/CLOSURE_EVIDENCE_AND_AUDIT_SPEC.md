# CLOSURE EVIDENCE AND AUDIT SPEC

## Purpose
This document defines the evidence and audit requirements for portfolio governance closure.

## Required closure evidence
Closure evidence must capture:

1. source-of-truth commit at closure
2. closure pack validation result
3. final governance state
4. portfolio clinic scope
5. closure criteria satisfaction result
6. evidence manifest
7. manifest hash

## Audit rule
The closure pack must remain traceable to the governed chain that preceded it.

## Evidence integrity rule
Closure evidence must be deterministic, committed, and auditable.

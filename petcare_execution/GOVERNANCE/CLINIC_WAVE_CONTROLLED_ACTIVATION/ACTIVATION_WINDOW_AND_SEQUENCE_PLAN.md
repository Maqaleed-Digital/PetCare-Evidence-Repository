# ACTIVATION_WINDOW_AND_SEQUENCE_PLAN

## Purpose
This document defines the controlled sequencing model for clinic activation windows.

## Sequence model
Clinics are activated in deliberate order, not in parallel without command oversight.

## Placeholder activation sequencing
The default deterministic sequence is:

1. CLINIC-WAVE2-001
2. CLINIC-WAVE2-002
3. CLINIC-WAVEN-001
4. CLINIC-WAVEN-002

## Window rule
Each clinic must have:

- an assigned activation window
- a command owner
- a first-day control checklist
- a hypercare entry record

## Resequencing rule
Portfolio command may resequence clinics if a dependency, safety, staffing, or reporting issue exists.

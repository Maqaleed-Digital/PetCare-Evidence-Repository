# FIRST_DAY_OPERATIONAL_CONTROL_SPEC

## Purpose
This document defines the minimum first-day controls required for a newly activated clinic.

## First-day controls
The following controls are mandatory on activation day:

1. human approval enforcement active
2. override capture active
3. escalation contact path reachable
4. first-day reporting snapshot enabled
5. command monitoring active
6. incident logging active
7. partner fallback path available
8. end-of-day control review scheduled

## First-day principle
The first day is treated as an evidence-rich supervised operating period, not normal steady-state operation.

## Failure rule
A material failure in any first-day control requires immediate command review and may trigger rollback to remediation or extended hypercare.

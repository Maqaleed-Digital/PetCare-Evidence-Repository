# PORTFOLIO COMPARISON SUMMARY FRAMEWORK

## Purpose
This document defines how executive comparison summaries are produced across the clinic portfolio.

## Required comparison summary domains
Executive comparison summaries must cover:

1. steady-state status by clinic
2. control posture by clinic
3. override posture by clinic
4. escalation posture by clinic
5. KPI completeness by clinic
6. drift posture by clinic
7. remediation posture by clinic
8. comparison against baseline clinic

## Summary rule
The baseline clinic remains the reference anchor for executive comparison.

## Required portfolio groupings
Comparison summaries must distinguish:

- baseline live clinic
- steady-state operational clinics
- clinics on watch
- clinics under remediation attention

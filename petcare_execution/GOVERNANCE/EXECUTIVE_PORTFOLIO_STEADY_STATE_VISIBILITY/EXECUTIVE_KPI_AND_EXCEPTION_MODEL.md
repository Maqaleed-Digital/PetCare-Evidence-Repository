# EXECUTIVE KPI AND EXCEPTION MODEL

## Purpose
This document defines the executive KPI and exception model for steady-state multi-clinic portfolio visibility.

## Required executive KPIs
The executive view must summarize:

1. clinic count in portfolio
2. steady-state operational count
3. clinics on watch count
4. open drift issue count
5. remediation reentry count
6. override posture summary
7. escalation posture summary
8. KPI completeness summary

## Required exception views
The executive layer must also surface:

- clinics requiring executive review
- clinics with drift issues open
- clinics under remediation reentry
- clinics under watch posture

## Rule
Executive KPI reporting is summary-first, but must remain traceable to governed portfolio reporting.

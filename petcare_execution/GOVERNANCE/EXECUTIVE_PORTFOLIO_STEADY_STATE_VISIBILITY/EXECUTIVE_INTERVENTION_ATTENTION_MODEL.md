# EXECUTIVE INTERVENTION ATTENTION MODEL

## Purpose
This document defines how executive intervention attention is classified across the steady-state clinic portfolio.

## Attention categories
Executive attention is classified as:

1. routine visibility
2. watch visibility
3. intervention attention
4. urgent portfolio review

## Attention triggers
Attention may be triggered by:

- control posture issue
- drift issue
- remediation reentry
- KPI completeness concern
- escalation posture concern
- cross-clinic comparison deterioration

## Rule
Executive attention does not replace governance authority. It prioritizes where executive review focus should be directed.

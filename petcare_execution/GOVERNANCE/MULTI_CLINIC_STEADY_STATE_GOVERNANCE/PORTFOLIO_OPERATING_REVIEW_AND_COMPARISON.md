# PORTFOLIO OPERATING REVIEW AND COMPARISON

## Purpose
This document defines the portfolio operating review model for comparing multiple steady-state clinics.

## Required review domains
Portfolio review must compare clinics across:

1. admission status
2. override posture
3. escalation posture
4. KPI completeness
5. partner fallback posture
6. staffing exception posture
7. drift posture
8. remediation exposure

## Review rule
Comparison is required between:

- baseline live clinic
- newly admitted steady-state clinics
- clinics on watch
- clinics under remediation review

## Review outcome use
The review model supports:

- portfolio operating decisions
- identification of underperforming clinics
- identification of control drift
- prioritization of remediation actions

# PORTFOLIO PERFORMANCE COMPARISON MODEL

## Purpose
This document defines the minimum comparison model for cross-clinic performance governance.

## Comparison principle
The reference clinic remains the baseline comparator. New clinics are not judged in isolation; they are assessed relative to approved operating norms.

## Minimum comparison metrics
1. activation status
2. hypercare status
3. override event rate
4. escalation event rate
5. KPI capture completeness
6. partner dependency concentration
7. staffing readiness coverage
8. AI safety attestation status

## Comparison buckets
Clinics are grouped into:

- baseline clinic
- newly activated clinics
- hypercare clinics
- steady-state clinics

## Governance use
The comparison model is used for:

- site admission decisions
- hypercare exit decisions
- remediation targeting
- wave replication planning
- portfolio governance reviews

## Required reporting outcome
The portfolio view must show whether each clinic is:

- below baseline
- within controlled range
- above baseline
- requires remediation review

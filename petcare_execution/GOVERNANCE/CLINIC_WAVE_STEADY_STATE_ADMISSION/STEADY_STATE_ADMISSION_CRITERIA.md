# STEADY_STATE_ADMISSION_CRITERIA

## Purpose
This document defines the minimum criteria required before a clinic may be admitted from hypercare into steady-state.

## Admission criteria
The following criteria are mandatory:

1. hypercare exit review completed
2. no unresolved material deviation
3. override posture within controlled tolerance
4. escalation posture confirmed operational
5. KPI completeness confirmed stable
6. partner fallback posture reviewed
7. portfolio comparison visibility active
8. steady-state control posture confirmed

## Admission outcomes
The only allowed outcomes are:

- admitted_to_steady_state
- remain_in_hypercare
- return_to_remediation

## Decision rule
Partial admission is not allowed. Admission is a single governance decision per clinic.

# HYPERCARE_OBSERVATION_CONTROL_MODEL

## Purpose
This document defines the minimum observation controls required while a clinic is in hypercare.

## Required observation controls
The following controls are mandatory during hypercare:

1. override event monitoring
2. escalation event monitoring
3. KPI completeness monitoring
4. staffing exception monitoring
5. incident review monitoring
6. partner fallback monitoring
7. command review cadence
8. AI safety posture review

## Observation rule
A clinic remains in hypercare while these controls are active and being reviewed.

## Observation states
Each clinic in hypercare must be visible as one of:

- under_observation
- observation_issue_detected
- observation_stable

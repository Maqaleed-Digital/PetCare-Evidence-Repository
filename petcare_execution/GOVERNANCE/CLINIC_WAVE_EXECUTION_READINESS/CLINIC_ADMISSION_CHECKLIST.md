# CLINIC ADMISSION CHECKLIST

## Purpose
This checklist defines the minimum conditions for admitting a clinic placeholder into wave execution readiness.

## Admission items
Each clinic must satisfy all items below:

1. clinic code assigned
2. wave assignment confirmed
3. local clinical lead assigned
4. staffing minimum confirmed
5. human approval boundary confirmed
6. override capture enabled
7. escalation contact chain assigned
8. reporting path enabled
9. partner dependency map recorded
10. evidence directory generation confirmed

## Admission rule
Admission is pass/fail. Partial admission is not allowed.

## Clinics in scope
- CLINIC-WAVE2-001
- CLINIC-WAVE2-002
- CLINIC-WAVEN-001
- CLINIC-WAVEN-002

## Required decision outcome
Each clinic must be explicitly classified as one of:

- admitted_to_execution_readiness
- blocked_pending_remediation

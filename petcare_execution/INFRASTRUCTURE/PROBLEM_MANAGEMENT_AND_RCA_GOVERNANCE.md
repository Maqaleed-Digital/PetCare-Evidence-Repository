PROBLEM_MANAGEMENT_AND_RCA_GOVERNANCE

Purpose
Define the problem management and root-cause governance model.

Problem Record Fields
- problem_id
- linked incidents
- owner
- severity
- affected service
- root cause summary
- containment summary
- corrective action
- preventive action
- due date
- closure status

Rules
- recurring incidents require problem review
- Sev-1 incidents require RCA or linked follow-up record
- corrective and preventive actions must have owners
- problem closure requires action completion or accepted exception

PETCARE-NATIONAL-SERVICE-NETWORK-AND-REGIONAL-OPERATING-GOVERNANCE
REGIONAL_OPERATING_UNIT_GOVERNANCE

Purpose
Define the governance model for each regional operating unit in the PetCare network.

Required Regional Fields
- region_id
- regional operating owner
- clinic set in region
- partner set in region
- support ownership
- capacity status
- risk status
- escalation coverage
- review cadence
- readiness status

Rules
- each region requires explicit ownership
- support and escalation ownership must be named
- regional readiness and risk must be reviewed on a named cadence
- no region is considered governed without a named owner and review structure

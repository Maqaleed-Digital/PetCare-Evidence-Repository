PETCARE-PRODUCTION-HYPERCARE-AND-OPERATIONS-GOVERNANCE
DAILY_OPERATIONAL_REVIEW_TEMPLATE

Purpose
Provide the daily review template for production hypercare operations.

Required Sections
- review_date_utc
- attendees
- source_of_truth_commit
- release version in production
- runtime health summary
- incident summary
- KPI and SLO summary
- audit path summary
- AI governance path summary
- clinic/pharmacy operational notes
- decision log
- follow-up owners

Rule
A daily operational review is complete only when all required sections are filled and owners are named for follow-up items.

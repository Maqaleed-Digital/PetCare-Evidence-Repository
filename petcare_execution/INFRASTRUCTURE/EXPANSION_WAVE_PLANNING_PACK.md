PETCARE-SCALE-EXPANSION-READINESS-AND-MULTI-CLINIC-EXECUTION-GOVERNANCE
EXPANSION_WAVE_PLANNING_PACK

Purpose
Define the planning pack for each scale expansion wave.

Required Wave Fields
- wave_id
- target clinics or nodes
- target region
- launch window
- launch owner
- readiness status
- dependency summary
- partner onboarding summary
- capacity summary
- risk summary
- approval status

Rules
- every wave requires explicit scope
- launch window must be named
- readiness and dependencies must be reviewed before approval
- wave approval must be captured before execution

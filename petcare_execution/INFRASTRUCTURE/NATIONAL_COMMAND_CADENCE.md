NATIONAL_COMMAND_CADENCE

Purpose
Define the national command cadence required for enterprise network orchestration.

Cadence Layers
- daily national operating signal review
- weekly national network coordination review
- monthly national optimization review
- quarterly enterprise governance review

Required Fields
- cadence_type
- meeting owner
- participants
- input packs
- escalation summary
- decisions and owners
- follow-up date

Rules
- each cadence requires named owner
- decisions require named owners
- unresolved critical blockers must appear in command review
- evidence references are mandatory

Scope

- executive scale review

Named Roles
- scale program owner
- clinic launch owner
- operations scale owner
- partner onboarding owner

Rules
- each clinic or node must pass readiness before launch
- expansion must proceed by explicit wave

- stabilization evidence is mandatory before declaring wave complete
- scale decisions must remain evidence-backed

Success Rule
Multi-clinic scale execution is governed only when the governance model, ownership, and wave controls are documented and linked to evidence.

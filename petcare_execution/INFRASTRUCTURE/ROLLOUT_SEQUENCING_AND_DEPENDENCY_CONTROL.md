ROLLOUT_SEQUENCING_AND_DEPENDENCY_CONTROL

Purpose
Define sequencing and dependency control for scale rollout.

Required Sequencing Elements
- wave order
- clinic order within wave
- blocking dependencies
- approval checkpoints
- stabilization checkpoints
- escalation path

Rules
- rollout must be sequenced explicitly
- dependencies must be visible before launch
- unresolved blocking dependencies prevent progression
- stabilization checkpoint must pass before next wave proceeds

DF35 — TRUST AND CERTIFICATION EVIDENCE MODEL

REQUIRED RECORD FIELDS

- trust_output_id
- partner_reference
- trust_state
- certification_state
- criteria_reference
- approval_requirement
- evidence_references
- validator_result
- timestamp

RULES

- every trust output must identify evidence basis
- every certification output must identify criteria reference
- every blocked output must still produce evidence
- no output may imply automatic certification

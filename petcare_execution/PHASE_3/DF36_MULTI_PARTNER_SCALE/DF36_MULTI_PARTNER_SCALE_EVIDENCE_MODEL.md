DF36 — MULTI-PARTNER SCALE EVIDENCE MODEL

- scale_output_id
- partner_cohort_reference
- scale_state
- isolation_reference
- approval_requirement
- evidence_coverage_reference
- evidence_references
- validator_result
- timestamp

RULES

- every scale output must identify cohort boundary
- every scale output must identify isolation reference
- every blocked output must still produce evidence
- no output may imply blanket enablement

# PETCARE DF17 — Prioritization Evidence Model

## Required Artifacts Per Prioritization Run
- priority_record.json
- scoring_inputs.json
- scoring_output.json
- ranking_snapshot.json
- decision_log.txt

## Required Artifacts Per Approved Override
- override_record.json
- previous_weights.json
- new_weights.json
- approval_log.txt

## Traceability Links
Each prioritization run must link to:
- DF15 backlog reference
- DF15 KPI baseline reference
- DF16 execution unit reference
- override reference if applicable

## Integrity
All artifacts must be included in MANIFEST.json with SHA256 digests.

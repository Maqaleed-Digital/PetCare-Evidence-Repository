# PETCARE DF20 — Policy Change Evidence Model

## Required Evidence Artifacts
- policy_change_record.json
- policy_before.json
- policy_after.json
- version_transition.json
- approval_log.txt
- impact_statement.txt
- decision_log.txt

## Traceability
Each policy change must link to:
- DF17 policy context if scoring-related
- DF18 policy context if capacity-related
- DF19 evidence basis if feedback-driven

## Integrity
All artifacts must be hashed and included in MANIFEST.json.

PILOT EVIDENCE CONTRACT

Required Evidence Outputs

- decision_log.txt
- execution_state.json
- operator_inputs.json
- checklist_status.txt
- blocked.log or active.log
- MANIFEST.txt
- git_head.txt

Blocked Path

Used when any required live variable is missing or contains forbidden placeholder content.

Active Path

Used only when all required live variables are present and valid.

Integrity Rule

Evidence must reflect the actual execution state only.
No simulated PASS output is allowed.

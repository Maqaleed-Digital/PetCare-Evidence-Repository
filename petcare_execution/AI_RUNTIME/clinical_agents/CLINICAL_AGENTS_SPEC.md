# Clinical Assistive Agents Specification

Pack: PETCARE-AI-INT-2

Agents Introduced

1. Clinical Copilot
   - summarizes patient case
   - highlights allergies
   - highlights medication conflicts

2. Prescription Safety Agent
   - checks medication safety
   - flags dosage anomalies
   - flags drug interactions

3. Triage Support Agent
   - assists triage classification
   - highlights emergency signals
   - proposes triage category

4. Pharmacy Intelligence Agent
   - detects low inventory
   - predicts restock needs
   - flags medication expiration

5. Emergency Coordination Agent
   - prioritizes emergency queue
   - assists emergency workflow coordination

Safety Enforcement

Agents cannot:
- prescribe medication
- authorize treatment
- finalize diagnosis
- close consultation
- sign clinical record

Human approval required for all clinical decisions.

DF40 Approved Abuse Prevention Control Catalog

APC-01
Name
Shadow discount detection control

Allowed When
Used to detect undeclared discounts, rebates, credits, or side arrangements that bypass approved commercial governance

Required Controls
rule reference
review owner
evidence trace
human review before restriction

APC-02
Name
Manipulative ranking exposure control

Allowed When
Used to detect undeclared commercial influence over visibility or discoverability

Required Controls
visibility audit
comparison class
approval reference
rollback posture

APC-03
Name
Referral steering abuse control

Allowed When
Used to identify commercial steering that overrides approved routing logic without declared governance

Required Controls
routing evidence
rule reference
review path
human-approved intervention only

APC-04
Name
Incentive stacking abuse control

Allowed When
Used to detect overlapping incentives that create unfair advantage or non-transparent opportunity concentration

Required Controls
cross-incentive review
fairness class check
approval reference
bounded response

APC-05
Name
Exclusivity coercion control

Allowed When
Used to detect partner pressure, forced exclusivity, or lock-in structures outside approved contracts

Required Controls
commercial evidence
partner treatment standard
appeal path
no autonomous penalty

APC-06
Name
Penalty proportionality review control

Allowed When
Used to ensure proposed commercial restrictions remain proportionate to the rule breach and reversible where applicable

Required Controls
reason code
impact review
approval reference
rollback rule

Universal Rule

No approved abuse-prevention control may execute penalties automatically. Every materially adverse action requires human review, evidence, and reversible posture.

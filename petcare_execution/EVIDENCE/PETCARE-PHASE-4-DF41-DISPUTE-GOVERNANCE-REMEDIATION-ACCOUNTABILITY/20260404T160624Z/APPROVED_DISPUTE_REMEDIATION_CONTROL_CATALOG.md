DF41 Approved Dispute and Remediation Control Catalog

ADC-01
Name
Structured dispute intake control

Allowed When
Used to register a dispute with classification, owner, contested action, and evidence references

Required Controls
classification ruleset
owner
audit trace
review mode

ADC-02
Name
Evidence preservation control

Allowed When
Used to preserve and link evidence across intake, review, appeal, and remediation

Required Controls
evidence list
trace continuity
tamper-aware documentation
human-reviewed closure

ADC-03
Name
Remediation assignment control

Allowed When
Used to assign corrective action following an approved review outcome

Required Controls
accountable owner
due date
evidence update requirement
rollback or restoration posture

ADC-04
Name
Appeal activation control

Allowed When
Used to re-open or escalate an in-scope dispute class through an approved appeal path

Required Controls
appeal reason
reviewer assignment
linked prior evidence
fresh decision record

ADC-05
Name
Resolution proportionality control

Allowed When
Used to ensure dispute outcomes and remediation requirements remain proportionate to the reviewed facts and approved standards

Required Controls
reason code
impact review
approval reference
reversibility posture

ADC-06
Name
Post-resolution accountability review control

Allowed When
Used to verify remediation completion, restored baseline, and closure legitimacy

Required Controls
completion evidence
review owner
closure note
post-resolution review timestamp

Universal Rule

No approved dispute or remediation control may autonomously close a case, assign fault, or execute materially adverse outcomes without human review and evidence-backed accountability.

Readiness Gap and Finding Register Standard

Purpose

Define how readiness gaps, open findings, and certification blockers are surfaced and governed.

Gap Classes

AG-01
Missing evidence link

AG-02
Pending runtime validation

AG-03
Documentary-only control

AG-04
Open implementation gap

AG-05
Unsupported readiness claim prevented

Required Gap Fields

1. readiness_gap_id
2. control_id
3. gap_class
4. current readiness status
5. owner
6. remediation path
7. target review date
8. closure evidence reference

Universal Rule

No readiness gap or finding may be suppressed, reclassified silently, or treated as closed without closure evidence.

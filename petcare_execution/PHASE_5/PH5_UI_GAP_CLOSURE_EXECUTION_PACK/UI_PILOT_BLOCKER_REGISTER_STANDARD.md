UI Pilot Blocker Register Standard

Purpose

Define how pilot-blocking UI gaps are recorded, prioritized, and governed until closure.

Required Fields

1. blocker_id
2. ui_gap_id
3. blocker_surface
4. blocker_class
5. current_status
6. pilot_impact
7. owner
8. remediation_path
9. target_validation_date
10. closure_evidence_ref

Pilot Impact Values

1. blocks_pilot_start
2. blocks_role_access
3. blocks_critical_workflow
4. blocks_auth
5. blocks_operational_visibility

Universal Rule

No pilot blocker may be hidden, downgraded silently, or represented as closed without closure evidence.
